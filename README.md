# 📄 GST Invoice Validator Bot

An RPA (Robotic Process Automation) project developed using UiPath to automate GST invoice validation and data extraction.

## 🚀 Features

* Reads GST invoices automatically
* Validates invoice details and GST information
* Extracts invoice data from PDF documents
* Uses OCR for scanned invoices
* Reduces manual effort and processing time
* Improves accuracy in invoice verification

## 🛠️ Tools & Technologies

* UiPath Studio
* Excel Activities
* OCR (Optical Character Recognition)
* PDF Automation
* Data Extraction

## 📋 Workflow

1. Read invoice PDF
2. Extract invoice details using OCR
3. Validate GST information
4. Store extracted data in Excel
5. Generate validation results

## 🎯 Benefits

* Faster invoice processing
* Reduced manual errors
* Automated GST validation
* Improved operational efficiency
